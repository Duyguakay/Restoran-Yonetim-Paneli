Proje: Restoran Yönetim Paneli
Geliştiriciler: Zeynep Duygu Akay & Çiğdem Bircan

Projemizi yerel bilgisayarınızda çalıştırabilmek için lütfen aşağıdaki adımları takip ediniz:

1. Klasör içerisinde yer alan "restoran_veritabani.sql" dosyasını MySQL Workbench (veya tercih ettiğiniz bir MySQL arayüzü) üzerinden "Import" ederek veritabanı şemasını ve gerekli tabloları otomatik olarak oluşturabilirsiniz.
2. "pass.env" dosyasının içerisindeki DB_PASSWORD alanına kendi yerel MySQL şifrenizi yazınız.
3. Proje klasörünün dizinindeyken terminalde "python3 app.py" komutunu çalıştırarak web arayüzüne (http://127.0.0.1:5000) erişim sağlayabilirsiniz.
